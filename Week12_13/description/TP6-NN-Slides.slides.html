Neural Networks¶

Neural Networks are versatile. Basic principle is to learn a function:

\begin{equation} f: X \in \mathbb{R}^N \rightarrow Y \in \mathbb{R}^P \end{equation}

Trained using optimization techniques (gradient descent with backpropagation)

Layers of activation functions¶

Layers of functions are stacked to learn complex representations:

\begin{equation} a_i^l = g(\sum\limits_{j=1}^{S^{l-1}} w_j^l.a_j^{l-1} + w_{0j}^l), \end{equation}

Using vectorization:

\begin{equation}\label{eq:forward} A^l = g(W^l.A^{l-1} + W_0^l), \end{equation}

Type of activation functions¶

Common activation functions include:

  • Sigmoid
  • tanh
  • ReLU (Rectified Linear Unit)
  • Leaky ReLU
  • RBF
